### ‚úÖ `floor()`

Returns the nearest whole number **less than or equal** to the given number.

```python
import math

math.floor(3.5)      # 3  
math.floor(3.9)      # 3  
math.floor(-3.5)     # -4  
math.floor(-2.8)     # -3  
```

---

### ‚úÖ `trunc()`

Cuts off the decimal part and moves the number **towards zero**.

```python
math.trunc(2.8)      # 2  
math.trunc(-2.8)     # -2  
```

## üîÅ Number Conversion

### ‚û§ Binary

```python
bin(2)          # '0b10'  
int('10', 2)    # 2  
```

### ‚û§ Octal

```python
oct(64)         # '0o100'  
int('100', 8)   # 64  
```

### ‚û§ Hexadecimal

```python
hex(64)         # '0x40'  
int('40', 16)   # 64  
```

---

## üî¢ Decimal

Used for exact decimal values (like money).

```python
from decimal import Decimal

Decimal('0.1') + Decimal('0.2')    # Decimal('0.3')  
```

---

## ‚ûó Fraction

Used for exact fractions (no float errors).

```python
from fractions import Fraction

Fraction(1, 3)          # Fraction(1, 3)  
Fraction('3/4')         # Fraction(3, 4)  
```

---

## üü∞ Set

Used to store **unique** items. Removes duplicates.

```python
a = {1, 2, 2, 3}         # {1, 2, 3}

{1, 2} | {2, 3}          # Union ‚Üí {1, 2, 3}  
{1, 2} & {2, 3}          # Intersection ‚Üí {2}  
{1, 2, 3} - {2}          # Difference ‚Üí {1, 3}  
```
