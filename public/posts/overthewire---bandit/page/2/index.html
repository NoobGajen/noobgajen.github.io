Bandit: Level 19 → Level 20

To gain access to the next level, you should use the setuid binary in the homedirectory. Execute it without arguments to find out how to use it. The password for this level can be found in the usual place (/etc/bandit_pass), after you have used the setuid binary. Here are the details which we have from the previous level: Host: bandit.labs.overthewire.org Port: 2220 Username: bandit19 Password: cGWpMaKXVwDUNgPAVJbWYuGHVn9zl3j8 So, The SSH syntax will be: sshpass -p cGWpMaKXVwDUNgPAVJbWYuGHVn9zl3j8 ssh bandit19@bandit.labs.overthewire.org -p 2220 We are dealing with a SUID binary. SUID or setuid, is a special type of permission in Unix-like operating systems that allows users to execute a file with the permissions of the file’s owner rather than their own permissions.
