‚úÖ `floor()`

January 13, 2026 ¬∑ 1 min

Returns the nearest whole number less than or equal to the given number.

import math

math.floor(3.5)      # 3  
math.floor(3.9)      # 3  
math.floor(-3.5)     # -4  
math.floor(-2.8)     # -3

‚úÖ trunc()#

Cuts off the decimal part and moves the number towards zero.

math.trunc(2.8)      # 2  
math.trunc(-2.8)     # -2
